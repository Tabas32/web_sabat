How to set up simple website with hugo

January 24, 2023 One-minute read
Marian Sabat
web
  1. install hugo
  • on arch sudo pacman -S hugo
  • on mint sudo snap install hugo
  1. check installed
  • arch: hugo version
  • mint: snap run hugo version
  1. create new project
hugo new site temp
cd temp
git init
  1. get theme from: https://github.com/luizdepra/hugo-coder
    git submodule add https://github.com/luizdepra/hugo-coder.git themes/hugo-coder

update config.toml with minimal template from ^
comment customCSS, customSCSS, customJS (this is throwing error)

  1. run website
    hugo server

  2. customize to my needs

  3. new content
    hugo new about.md
    hugo new posts/first_post.md

  4. deploy in github pages tbd (treba nastavit github actions a zmenit settings v githube)
